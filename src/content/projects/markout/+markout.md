A command-line diff tool built in **C++** that strips empty lines and comments before comparing files, then highlights only meaningful changes in red. Single command, instant output, no GUI required.

Designed for the specific pain of getting a file from a collaborator and having no quick way to see what actually changed — without wading through diff noise from blank lines and stale comments.

Read the [blog post](/blog/markout) for the full write-up.
